# Frontend

Modern React-based E-commerce UI.

## Tech Stack
- **Language**: JavaScript (ES6+) / Node.js 20
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Modern UI)

## Dependencies
- `axios`: For API calls.
- `react-router-dom`: For client-side routing.
- `context-api`: For global state management (Auth, Cart).

## API Integration
- All API calls are made to `/api/*` via the **API Gateway**.
- Includes an Axios interceptor for JWT authentication.

## Running Locally
```bash
npm install
npm run dev
```
